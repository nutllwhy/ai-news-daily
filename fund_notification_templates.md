# 基金监控通知模板示例

## 1. 盘中提醒（下午1点，有重大新闻时发送）

**触发条件**：重大板块新闻或市场异动（±3%以上）

**示例格式**：
```
📢 【基金盘中提醒】2026-02-01 13:00

⚠️ 触发原因：科技板块重大政策新闻

📰 新闻摘要：
今日午间，工信部发布《人工智能产业发展规划（2026-2030）》，
明确提出对AI芯片、大模型等核心领域给予税收优惠和资金支持。

📊 涉及你的持仓：
• 国泰半导体设备ETF（持仓2.67万）
• 易方达人工智能ETF（持仓0.84万）
• 永赢科技智选（持仓0.88万）
• 国泰科创板芯片ETF（持仓2.03万）
合计：6.42万（占持仓22%）

💡 影响分析：
政策利好可能推动科技板块短期上涨，你持仓的科技基金可能受益。

⏰ 操作建议（仅供参考）：
1. 若下午开盘后科技板块大幅上涨，可考虑部分止盈
2. 若涨幅不大，可持有观察
3. 具体操作请根据你的风险偏好决定

⚠️ 提醒：以上仅为信息参考，不构成投资建议
```

## 2. 常规日报（晚上9点，每日必发）

**示例格式**：
```
📊 【基金日报】2026-02-01

💰 今日概况：
• 总持仓：29.05万
• 预估今日盈亏：+1,850元（+0.64%）
• 累计收益：+20.85%

🔔 提醒触发：
• 华夏有色金属：涨5.2%（接近止盈线55%）
• 恒生互联网：跌2.1%（接近补仓线-8%）

📈 板块表现：
• 科技：+1.2%
• 黄金：+0.8%
• 港股：-1.5%
• 医疗：+0.3%

📰 相关新闻：
1. 科技板块：AI产业政策利好...
2. 黄金：国际金价创新高...
3. 港股：恒指今日调整...

💡 数据参考：
• 大盘：沪深300 +0.5%，创业板指 +1.2%
• 你的科技板块占比：50.9%（偏高集中度）
```

## 3. 周报（每周日晚上9点，每周必发）

**示例格式**：
```
📈 【基金周报】2026年1月第4周

本周收益：+3.2%（+9,536元）
累计收益：+20.85%（+60,570元）

板块轮动分析：
• 强势板块：周期股（有色+化工）本周领涨
• 弱势板块：港股科技持续调整
• 建议关注：AI产业链政策动向

本周操作记录：
• 无（等待你的操作指令）

下周关注要点：
1. 美联储议息会议（影响美股QDII）
2. 国内PMI数据公布
3. 年报披露季开始

持仓调整建议（仅供参考）：
• 黄金收益较高，可考虑部分止盈
• 港股科技持续低迷，关注反弹机会
```

## 通知规则总结

| 时间 | 类型 | 触发条件 | 内容 |
|------|------|---------|------|
| 13:00 | 盘中提醒 | 有重大新闻/异动 | 新闻摘要+影响分析+涉及持仓 |
| 21:00 | 日报 | 每日必发 | 净值+收益+提醒+新闻+数据 |
| 周日21:00 | 周报 | 每周必发 | 周收益+板块分析+下周展望 |

**重要原则**：
- 盘中提醒：只发重要的，避免频繁打扰
- 所有建议都标注"仅供参考"
- 最终决策权完全在你
