【2026-02-14 栗噔噔AI早报】

各位早上好！今天是2026年情人节，也是AI界的“春晚”。Google、OpenAI、Anthropic 三巨头同日发布重磅更新，而 DeepSeek 则陷入了 OpenAI 的窃密指控风暴中。

🚨 硅谷「核爆日」
1. Gemini 3 Deep Think V2：理科生的终极形态
Google DeepMind 发布的 Gemini 3 Deep Think V2 在 ARC-AGI-2 基准测试中达到 84.6% 准确率，刷新 SOTA。其在 Codeforces 全球排名第 8，不仅能解题，还能辅助半导体研究和生成 3D 打印模型。

2. OpenAI × Cerebras：天下武功，唯快不破
OpenAI 推出与 Cerebras 合作的 GPT-5.3-Codex-Spark，生成速度突破 1000 tokens/s，比前代快 10 倍。软件开发的瓶颈正式从“写代码”转移到了“读代码”和“提需求”。

3. Anthropic 估值破 3800 亿，营收涨 10 倍
Anthropic 完成 300 亿美元融资，估值飙升至 3800 亿美元。其年收入突破 140 亿美元，相比去年增长 10 倍，Claude Code 产品 ARR 翻倍。与此同时，OpenAI 指控 DeepSeek 通过“蒸馏”窃取其数据。

💬 行业风向
4. 智谱 GLM-5 参数曝光：744B 巨兽
智谱 AI 的 GLM-5 模型参数量高达 7440 亿，训练数据量 28.5T tokens，集成了 DeepSeek 的稀疏注意力机制。虽然 Code Arena 排名第一，但在 Agent 任务上仍落后于 Claude Opus 4.6。

5. MiniMax M2.5 宣称对标 Claude Opus
MiniMax 推出 M2.5 模型，号称 SWE-Bench Verified 达到 80.2%，与 Claude Opus 持平。该模型主打“超长程 Agent”，凭借 100 tokens/s 的速度和低价策略进军开发者市场。
